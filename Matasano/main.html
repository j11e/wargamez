Each problem is in its own file, and the functions needed are declared in the same file, with the exception of the common conversion functions (bytes <-> hex <-> ascii <-> base64) and the output display function, which are put in misc.js. ALl the variables for each problem have their name prefixed with "chN_" so that they don't get overwritten, so they can still be played with in the console.
